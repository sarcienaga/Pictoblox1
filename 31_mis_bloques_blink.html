3.1. Mis Bloques Blink

Creando bloques

Como lo hicimos anteriormente, vamos a crear nuestro bloque para el blink del led, conectado en el pin digital 13, de la placa Arduino Nano de PonchoBot.

Creamos un nuevo bloque con el nombre Blink, para pegar debajo de él los bloques correspondientes al blink del led, que ya tenemos programados.

En el lugar que quedó libre (debajo de  siguiente disfraz )arrastramos el bloque Blink y debajo de él los bloques de  rebotar ...  y  esperar ...  para completar la programación.

    A experimentar!!

    PonchoBot trae incorporado un buzzer pasivo (pequeño parlante) conectado al pin 5 de Arduino Nano. Es posible generar un beep, por ej. cada vez que Tobi toque un borde. Para ello hay que usar un bloque ubicado en la categoría Arduino Nano

    Hay que saber que los valores 0 y 255 son para finalizar el beep, cualquier otro número entre ellos emitirá un beep; en nuestro caso usaremos el valor 111.

    Lea y complete

    • Al poner el valor del pin 13 en alto ( JXUwMDEwJXUwMDAxJXUwMDBlJXUwMDBm ), el led queda encendido durante el tiempo indicado por el bloque JXUwMDNkJXUwMDE2JXUwMDAzJXUwMDE1JXUwMDE3JXUwMDEzJXUwMDEz .
    • Al poner el valor del pin 13 en bajo ( JXUwMDE0JXUwMDAzJXUwMDE4 ), el led se apaga por un tiempo JXUwMDMxJXUwMDA3JXUwMDBhJXUwMDAxJXUwMDExJXUwMDExJXUwMDE3JXUwMDFmJXUwMDA0JXUwMDA3JXUwMDBmJXUwMDA1JXUwMDBi .
    • El valor 255 de set PWM ... indica que el beep del buzzer está JXUwMDM5JXUwMDExJXUwMDExJXUwMDA2JXUwMDA2JXUwMDA1JXUwMDBi .
    • El valor 111 de set PWM ... indica que el beep del buzzer está JXUwMDI4JXUwMDAyJXUwMDE3JXUwMDBiJXUwMDBhJXUwMDBkJXUwMDBkJXUwMDBi  durante el tiempo indicado por el bloque JXUwMDNkJXUwMDE2JXUwMDAzJXUwMDE1JXUwMDE3JXUwMDEzJXUwMDEz .
    • El valor 5 de set PWM ... indica el pin de JXUwMDNiJXUwMDBjJXUwMDAxJXUwMDBiJXUwMDFkJXUwMDExJXUwMDlhJXUwMDlk del buzzer pasivo a la placa Arduino Nano.

    Para verificar las respuestas clic en el botón Comprobar, para ver las respuestas correctas clic en el botón Mostrar retroalimentación.

    Habilitar JavaScript

    • Al poner el valor del pin 13 en alto ( HIGH ), el led queda encendido durante el tiempo indicado por el bloque esperar .
    • Al poner el valor del pin 13 en bajo ( LOW ), el led se apaga por un tiempo indeterminado .
    • El valor 255 de set PWM ... indica que el beep del buzzer está apagado.
    • El valor 111 de set PWM ... indica que el beep del buzzer está prendido  durante el tiempo indicado por el bloque esperar .
    • El valor 5 de set PWM ... indica el pin de conexión del buzzer pasivo a la placa Arduino Nano.
